知乎仿写--微信小程序
==================
- 原项目地址：https://github.com/RebeccaHanjw/weapp-wechat-zhihu
- 在原项目的基础上我增加了一些自己的想法，将语法精简，大量使用es6的语法糖。

![](./images/v_index.png)

#### 功能及使用的组件等
* 列表式渲染数据
* 自定义顶部 tabbar
* 下拉刷新
* 上拉加载更多
* 轮播图
